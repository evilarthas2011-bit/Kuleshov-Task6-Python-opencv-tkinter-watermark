# Наложение водяного знака и размытие фона (Tkinter + OpenCV)

Лабораторная работа по теме «Интеграция Tkinter, OpenCV и ООП». Вариант 12.

## Описание архитектуры
Проект реализован на языке Python с использованием объектно-ориентированного подхода (ООП) и разделен на два основных класса:
1. `App(tk.Tk)` — отвечает за построение графического интерфейса пользователя (GUI), создание двух вкладок («Оригинал» и «Обработка»), элементов ввода текста `Entry`, выбора степени размытия `Combobox` и кнопок управления.
2. `VisionProcessor` — инкапсулирует логику работы с компьютерным зрением (OpenCV): считывание матрицы изображения, динамический расчет геометрического центра кадра для точного позиционирования, размытие фона методом `cv2.blur` и нанесение текста водяного знака с помощью `cv2.putText`.

## Установка зависимостей
Для работы приложения требуются библиотеки OpenCV и Pillow. Установите их командой:
```bash
pip install opencv-python pillow
```

## Инструкция по запуску
Запустите главный скрипт программы из папки проекта:
```bash
python #TASK6.py
```
*(При необходимости замените `#TASK6.py` на точное имя вашего файла).*
